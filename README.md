# Droid - SeekArc
SeekArc Xamarin

Xamarin Android C# Port of [SeekArc](https://github.com/neild001/SeekArc/) 

The library provides a circular slider/seeker for Xamarin Android with fully customizable dimensions, angles, colors etc.
